Software

In this page we outline the different options to use iPython, and setup our notebook.

Selection of iPython interface

Using either

  • Google's Colaboratory
    • Login to your own Google account.
    • Click on File -> New Python 3 Notebook
    • Complete tasks below.
    • When Finished, click on File -> Download .ipynb to save notebook locally ready to submit for grading on Moodle.
  • Jupyter Lab
    • Click on Try JupyterLab which will start up a Binder instance
    • Close the two open demo notebook (on Lorenz DE and JuypterLab documentation).
    • Click on Python 3 notebook.
    • Complete tasks below.
    • When Finished, click on File -> Save Notebook and in the file viewer pane on the left, right click on your notebook (default name is Untitled.ipynb) and select Download to save notebook locally ready to submit for grading on Moodle.
  • Or your own laptop, running your own python installation. I prefer anaconda but other distributions work also.

Of the online options above, Google Colaboratory is better in that it appears to be more responsive and more importantly has persistent storage (if you use a Google account) --- you don't want to lose your work due to a broken connection.
